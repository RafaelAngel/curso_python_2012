Introducción¶

En esta sección se explica como instalar Python en Windows, Linux y Mac. Después se muestra como utilizar un entorno integrado de programación, orientado a la computación científica y parecido a Matlab, llamado Spyder. Finalmente se introducen las principales características del lenguaje y se busca que el estudiante se familiarice con ellas a través de una serie de ejercicios.
